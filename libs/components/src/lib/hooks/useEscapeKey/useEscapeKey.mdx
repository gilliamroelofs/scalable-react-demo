## useEscapeKey

- [How it works](#how-it-works)

### How it works

A react hook to check whether the user has pressed the escape key. It takes a handler function and a boolean to determine whether the hook should be active or not.

````jsx
useEscapeKey(handler, isActiveBoolean)
````
