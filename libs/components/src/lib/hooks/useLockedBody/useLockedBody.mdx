### How it works

A react hook to block the scrolling of the page.

An example of a use case is when you need to open a modal.

````jsx
export default function Component() {
  const [locked, setLocked] = useLockedBody(false)

  const toggleLocked = () => {
    setLocked(!locked)
  }

  return <Button onClick={toggleLocked}>{locked ? 'unlock scroll' : 'lock scroll'}</Button>
}
````
